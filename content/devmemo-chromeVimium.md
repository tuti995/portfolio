Vimium を入れたので、とりあえず覚えたいものをメモ。

```
? →　キー操作のヘルプを表示
j →　下スクロール
d →　半画面分下にスクロール
k →　上スクロール
u →　半画面分上にスクロール
f →　リンクのショートカットを表示
J →　一つ左のタブを表示する
K →　一つ右のタブを表示する
H →　遷移履歴を戻る
L →　遷移履歴を進む
t →　新規のタブを作る
x →　現在のタブを閉じる
X →　閉じたタブを復元する
yy →　ページのアドレスをコピー
gu →　表示ページよりも１つ上のURL階層に遷移する
gi →　入力ボックスにフォーカスする
```
