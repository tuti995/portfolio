URL について学んだ時のメモ

```
https://www.〇〇.co.jp/〇〇/index.html
```

```
https://  スキーム名
www.〇〇.co.jp  FQDN
www　ホスト名
〇〇.co.jp　ドメイン名
/〇〇/index.html　パス名
```

### scheme

- HTTP や HTTPS などのプロトコル
  - SSL 通信ではポート番号と呼ばれる番号を利用する
    - FTP 20 番（データ転送用）
    - SMTP 25 番
    - HTTP 80 番
    - HTTPS 443 番
